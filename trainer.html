Trainable model

A neural network you can watch learn.

This page trains a classifier from scratch — no ML libraries — using the same machinery that underlies large language models: a forward pass, a binary cross-entropy loss, and gradient descent that nudges the weights, epoch by epoch, toward predictions that match reality. Load a blood-work dataset, press Train, and watch the loss fall and the weights settle.

How this maps to training a large language model

The model here is small, but the training procedure is the same one used to train neural networks of every size, including LLMs. Logistic regression is literally a single-neuron network with a sigmoid activation; flip on the hidden layer and you have a genuine multi-layer net trained by back-propagation.

LLM trainingThis project
Billions of weightsOne weight per blood-work marker (+ a bias), or a small weight matrix with the hidden layer
Forward pass: tokens → logits → softmax → probabilitiesBlood-work → weighted sum → sigmoid → probability of diabetes
Cross-entropy loss on the next tokenBinary cross-entropy on the diagnosis — the same loss family
Back-propagation + an optimizer stepHand-written gradients + w ← w − η·∇, every epoch

A caveat worth stating, because it's the most interesting part: a model only ever learns what is in its data. This dataset shows HDL cholesterol rising with diabetes — the opposite of the usual clinical relationship — so the model dutifully learns that "backwards" signal. That is exactly the bias problem LLMs face with their training corpora: the optimizer faithfully fits whatever pattern the data contains, correct or not.

Not a medical device. This is a from-scratch machine-learning demonstration built on a public dataset of unverified origin. It does not diagnose anyone and must not be used for medical decisions.

Logistic regression / a one-hidden-layer neural network, trained with full-batch gradient descent and binary cross-entropy. Features are standardized on the training split only; evaluation is on a held-out test split. Everything runs client-side.
